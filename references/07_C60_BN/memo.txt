Below we provide an example input file (“brush.in”) that 
models a charge-regulating polyelectrolyte brush with 
epsilon_mm = kBT and pKa = 1 in a water solution characterized by 
a Bjerrum length lB = 0.72 nm and pH=7. The other simulation parameters 
can be found in the main text.
